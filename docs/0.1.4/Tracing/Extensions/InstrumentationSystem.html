  • tracer

    Returns the Tracer bootstrapped as part of the InstrumentationSystem.

    If the system was bootstrapped with a MultiplexInstrument this function attempts to locate the first tracing instrument as passed to the multiplex instrument. If none is found, a NoOpTracer is returned.

    Declaration

    Swift

    public static var tracer: Tracer { get }

    Return Value

    A Tracer if the system was bootstrapped with one, and NoOpTracer otherwise.
