  • subscript(_:)

    Accesses the SpanAttribute with the given name for reading and writing.

    Please be cautious to not abuse this APIs power to read attributes to “smuggle” values between calls. Only Baggage is intended to carry information in a readable fashion between functions / processes / nodes. The API does allow reading in order to support the subscript-based dynamic member lookup implementation of attributes which allows accessing them as span.attributes.http.statusCode, forcing us to expose a get operation on the attributes, due to the lack of set-only subscripts.
